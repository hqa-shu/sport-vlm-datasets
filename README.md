## 部署方式

本仓库通过 GitHub Pages 部署，单文件 `index.html` 无任何依赖（无 CDN、无 JS 库、无外部资源），离线可用。

### 本地预览

```bash
# 方式 1：直接双击 index.html
open index.html

# 方式 2：起本地服务器
python3 -m http.server 8000
# 访问 http://localhost:8000
```
